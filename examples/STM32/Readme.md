# STM32例程

​	本文档介绍了如何使用stm32单片机读取Hi226/Hi229的数据，本路径提供了c语言例程代码，生成的__.hex__文件用于烧写到单片机中，读取数据。

​	测试环境：Windows10_x64 操作系统

​	编译器：keil_V5.28

​	开发板：正点原子-战舰V3 STM32F103ZET6

​	测试设备：HI226	HI229

### 硬件连接

​	将Hi226/Hi229 正确插入到超核调试板上。

| 超核调试板 | 正点原子开发板 |
| ---------- | -------------- |
| RXD        | PA2(TXD)       |
| TXD        | PA3(RXD)       |
| 3.3V       | 3V3            |
| GND        | GND            |

​	用杜邦线将上表相对应的引脚连接起来。

​	用USB线插到开发板的 __USB_232__ 插口上，另一端插到电脑上。
